# 🏥 Doctors Management System

A robust Clinic Management System built to streamline patient data, appointments, and prescription tracking using modern .NET technologies.

## ✨ Features

* **Patient Management:** Seamless registration, structured history, and tracking of patient records.
* **Prescription System:** Efficient creation, management, and tracking of digital medical prescriptions.
* **Data Optimization:** Server-side pagination to process 10,000+ records smoothly with minimal memory.
* **Analytical Dashboard:** Built with WPF to dynamically visualize critical daily medical and financial KPIs.

## 🚀 Tech Stack

* **Backend:** .NET 8 / ASP.NET Web API
* **Desktop UI:** WPF (with WPF-UI for a modern Mica-enabled experience)
* **Database:** Entity Framework Core (SQL Server)
* **Architecture:** Onion Architecture / Clean Architecture principles
