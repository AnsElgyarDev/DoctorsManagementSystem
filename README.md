Doctors Management System
A clinic management system designed to streamline patient data and prescription tracking. Built with modern .NET technologies.

🚀 Tech Stack
Backend: .NET 10 Web API

Desktop: WPF (WPF-UI library)

Database: Entity Framework Core (SQL Server)

Architecture: Clean Architecture principles

🛠️ Features
Patient registration and management.

Prescription tracking.

Modern, production-ready UI with Mica support.

📋 Getting Started
Clone the repository.

1. Configure your connection string in appsettings.json.

2. Run Entity Framework migrations to set up the database.

3. Launch the Desktop application.
