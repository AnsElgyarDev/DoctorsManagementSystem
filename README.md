# 🏥 Doctors Management System

A robust Clinic Management System built to streamline patient data, appointments, and prescription tracking using modern .NET technologies.

## 🚀 Tech Stack

- **Backend:** .NET 10 Web API
- **Desktop UI:** WPF (with WPF-UI for a modern Mica-enabled experience)
- **Database:** Entity Framework Core (SQL Server)
- **Architecture:** Clean Architecture

## ✨ Features

- **Patient Management:** Seamless registration and tracking of patient records.
- **Prescription System:** Efficiently manage and track medical prescriptions.
- **Modern UI:** Clean, intuitive interface with a focus on usability and performance.

## 🛠️ Getting Started

To get this project running locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/AnsElgyarDev/DoctorsManagementSystem.git](https://github.com/AnsElgyarDev/DoctorsManagementSystem.git)
   ```

Configuration:

1. Update the connection string in appsettings.json to match your local SQL Server instance.

2. Database Setup:
   Run the following command in the Package Manager Console:
   Update-Database

3. Run:
   Build the solution in Visual Studio and launch the Desktop application.
